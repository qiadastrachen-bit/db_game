# 碎梦童话 · 第一期「封面之野」

2D 像素规则怪谈叙事解谜网页游戏。原作/署名：**吃鸡腿的鸡腿姐🐧**。气质参考 Undertale + Omori。

- 技术栈：Phaser 3 + TypeScript + Vite
- 逻辑分辨率 480×270，CSS `image-rendering: pixelated` 整数倍放大（×1/×2/×3 自适应）
- UI 层（UIScene）与场景层（CoverFieldScene 等）分离，并行运行
- 全部叙事/律令/结局文本 JSON 驱动（`src/data/*.zh.json`、`i18n/zh.json`），不硬编码进 TS 逻辑

## 快速开始

```bash
npm install
npm run dev          # 开发服务器
npm run dev -- --port 7100          # CLI 参数直接转发给 Vite（--host 同理）
npm run build        # 产物在 dist/，相对路径打包，可任意静态部署
npm run typecheck    # tsc --noEmit，应 0 错误
npm run smoke        # 冒烟检查（需先 build；纯静态检查，不启动服务器）
```

## 本期验收清单

### 流程

1. **Boot → Menu**：打开页面，加载后出现标题「碎梦童话」与 开始/读档/图鉴/设置/制作名单。
2. **序章**：点「开始新的翻阅」→ 6 屏塑界之书原文逐屏打字机展示，E/点击翻页，Esc 可跳过；最后一屏点「翻 开」进入封面之野。
3. **封面之野**：开场 5 行旁白对话（E 推进），之后可自由移动。

### 五条律令（全部硬惩罚；未读规则前危险动作只弹经卷不结算，属公平性设计）

| # | 律令 | 经卷触发方式 | 违反方式（规则入图鉴后） |
|---|---|---|---|
| 1 | 午夜时勿踩哽咽植被 | 玩家走到 x≥240 触发午夜事件 → 对话后自动弹经卷 | 午夜后踩进灰绿色、颤抖的麦草丛（半径内） |
| 2 | 严禁触碰残月角 | 玩家走到 x≥320 自动弹经卷 | 鼠标点击夜空右上角的残月碎片 |
| 3 | 兔子眼睑静止不眨时必须停止移动 | 兔子首次出现并凝视时 → 对话后弹经卷 | 兔子凝视期间（450ms 宽限后）继续移动 |
| 4 | 请不要看指示牌 | 走近指示牌（64px 内）自动弹经卷 | 走近后按 E 或点击指示牌 |
| 5 | 请小心看路 | 走近路面爪印（40px 内）自动弹经卷 | 踩上小路上的暗红爪印花纹 |

### 两个结局

- **坏结局 `end_cover_claw`**：违反上表任意一条 → 低音 + 震屏 + 黑幕 → 结算页「你的存在不再归属于活物。」+ 爪印盖落静帧（无血腥）→ 记入结局画廊。
- **通行结局 `end_cover_pass`**：全程不违规，沿小路走到最右端发光的「页缝」（y 168~208 之间）→ 通行对话 → 白淡出 → 结算页「你仍是『你』。」+ 下一境预告 → 记入结局画廊。
- 结局画廊验证：回主菜单 → 图鉴 → 结局画廊页签，已解锁显示横幅+描述，未解锁显示爪印剪影。

### 系统验证

- **S 快速存档**：游戏中按 S → 顶部 Toast「已存入快速存档」+ 确认音效；回主菜单 → 读取存档 → 第一行「快速存档 · 封面之野 · 时间」，读取后位置/午夜状态/图鉴一致恢复。
- **Esc → 写入存档**：暂停菜单选 写入存档 → 选档位 1/2/3 → Toast 确认；读档面板可见对应档位。
- **游戏内读档**：Esc 暂停菜单 → 读取存档 → 列出 快速存档+档位 1-3（空档置灰）→ 点击后黑场切换，位置/flags/午夜状态正确恢复；Esc 可关闭该面板返回暂停菜单。
- **面板退出**：主菜单 读档/图鉴/设置 各面板右上角均有「✕ 关闭」可点击按钮，面板下方有「Esc 返回」提示，Esc 与点击均可退出。
- **J 律令图鉴**：已读律令显示原文+阐释（多行自动断行），未读显示爪印剪影「尚未翻到此页」，计数 x/5；J 或 Esc 合上。
- **图鉴跨档保留**：解锁结局/读过的律令在 开始新的翻阅 后仍保留（存 `localStorage` 图鉴键）。
- **设置**：主菜单或暂停菜单 → 设置，三条音量滑条实时生效并持久化（刷新页面后保留）。

### 显示与排版验证（第二轮修复）

- **序章换行**：序章长段落（第 2/4 屏）在纸页框内多行居中显示，不溢出屏幕左右边缘。
- **画面居中**：任意调整浏览器窗口大小，画布始终水平垂直居中，四周黑边对称，像素仍为整数倍放大。
- **经卷换行**：一律令 1（最长标题+正文）为例，经卷弹窗标题居中、正文多行显示，不超出弹窗边框；J 图鉴内已读律令的阐释同样多行显示。
- **菜单/结局文本**：图鉴结局描述、结局页描述均自动换行。
- **文字锐度**：×2/×3 放大下，序章正文、经卷、对话框、菜单按钮文字边缘锐利无虚化（图像素材保持像素锯齿为预期风格）。

## 已知限制

- 浏览器自动播放策略：首次点击/按键后才有声音（属预期行为）。
- 封面之野为直线关卡，无遮挡寻路（点击移动为直线走向目标）。
- 律令图鉴在菜单中只显示原文，完整阐释文本在游戏内 J 面板查看。
- 渲染采样策略（验收红线）：图像素材纹理 = NEAREST（像素边锐利），文字纹理 = LINEAR + resolution ceil(显示缩放) 超采样（非整数窗口下平滑不碎裂）；canvas CSS 尺寸锁定为 480×270 整数倍、居中留黑边，不使用 CSS pixelated。
- 点击防穿透：对话/经卷/面板打开期间世界层不响应点击移动；UI 关闭后 150ms 内的首次点击视为 UI 点击（防同一帧穿透），之后正常。
